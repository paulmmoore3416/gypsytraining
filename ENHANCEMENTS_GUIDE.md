## üó∫Ô∏è Enhancement 7: Behavior Tracking Heatmap

### Purpose
Visual representation of behavior patterns over 30-day periods.

### Features
- 30-day rolling heatmap
- Color-coded intensity levels
- Multiple behavior types
- Pattern recognition
- Daily notes tracking

### Behavior Categories
- Anxiety episodes
- Hypervigilance events
- Calm/grounded periods
- Alert responses
- Nightmare interruptions
- Deep pressure therapy
- Other custom behaviors

### How to Use
1. Go to **Behavior Analysis** section
2. Select behavior type
3. Set intensity (1-10 slider)
4. Add optional notes
5. Click **"Log Behavior"**
6. View heatmap:
   - Light colors = low intensity/frequency
   - Dark colors = high intensity/frequency
   - Patterns emerge over time

### Color Scale
- Gray: No activity
- Blue: Low intensity (1-4)
- Gold: Medium intensity (5-7)
- Orange: High intensity (8-10)

### Benefits
- Identify triggers
- Track improvement
- Recognize patterns
- Inform VA appointments
- Adjust training focus

## üìû Support & Troubleshooting

### Common Issues

**Certificate won't print:**
- Check print preview
- Ensure pop-ups aren't blocked
- Try "Download/Print" button

**Videos won't upload:**
- Check file size (<100MB)
- Verify video format (MP4, MOV)
- Ensure browser storage available

**Charts not displaying:**
- Click "Refresh Analytics"
- Check if training data exists
- Clear browser cache if needed

**Notifications not working:**
- Grant permission when prompted
- Check browser notification settings
- Ensure site isn't muted

**Heatmap missing data:**
- Verify behavior logs exist
- Check date range
- Reload page
